Project Details

Goal: Create a more user friendly website and implement responsive design.

Role: I designed mock-ups of the new webpages and layouts, created working prototypes of the mobile friendly version to use as the base for coding, coded the HTML, stylized the CSS, and ensured fluid responsive design.

Features: Added responsive design, added a mobile friendly menu navigation, re-designed certain features to be more user friendly, and created new layouts that help the user move through the website easier.

Challenges: The current version of the website was not responsive and many elements were not functioning properly. By creating a more user friendly layout and using clean coding, I was able to transition the design to a responsive layout and ensure all elements were correctly functioning.
