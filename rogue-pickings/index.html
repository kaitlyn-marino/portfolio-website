Rogue Pickings

The client provided a PSD mock-up file for the website design and requested a static home page (non-responsive). I built the webpage using HTML and CSS, implementing clean code and commenting for easy future updates and troubleshooting.

PSD Conversion: Built the webpage using HTML and CSS to match the PSD file provided. Images were extracted, sized accordingly, and saved using file naming convention best practices.

Display: Inline-Block: Implemented a three column layout using CSS inline-block properties. Using the inline-block property and floating the elements properly, I was able to match the exact layout provided in the mock-up.

Horizontal Navigation Menu: Properly aligned the menu navigation items horizontally to mimick the mock-up file. Exact spacing between menu items were added by using the ruler feature in Photoshop and stylizing the CSS.

Stylized Content: Used div classes and span tags in the HTML file to allow for easy and clean CSS style updates. This in turn allowed for different fonts and font sizing within the same line of text.
