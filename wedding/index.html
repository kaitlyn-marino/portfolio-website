Collin & Kaitlyn's Wedding

I designed mock-ups and built working prototypes of each page matching the wedding colors and branding from the Save the Dates and Wedding Invitations.

After the prototyping stage was complete, I coded the website using clean code and comments.

Responsive design elements were implemented for an exceptional user experience on all platforms. JavaScript was programmed for interactive elements, enhancing the user experience even further.

Full Screen Hero Image: Designed the desktop version of the home page to be a full screen hero image with a navigation menu bar for website simplicity and user experience.

Flexbox Layouts: Implemented layouts using CSS Flexbox to allow for single column layouts on smaller screens and double column layouts on larger screens. Flexbox was also used for content order consistency between column layouts.

Forms and JavaScript: Implemented fillable forms for wedding guests to rsvp. Utilized JavaScript to import an excel sheet of the guest list to auto-populate other members of their party for an enhanced user experience. JavaScript was also used to export the data into a file for easy data use.

Iframes & Directions: Iframes were used to enhance the user experience and easily show the location of the venue. Google Maps references were added through a directions button to ensure the wedding guests could easily populate directions to the venue.
