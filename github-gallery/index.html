Project Details

Goal: Create a page to display the latest repos on GitHub and give a high level overview of each.

Role: Created the JavaScript to pull the current information from GitHub to display on the web page and created a search box to search by repo.

Features: Pulls in the name, bio, location, and number of repos, added a search box to search through repos, when the repo is clicked it displays the description of the project, the default branch, and the languages used in the project, and has links from each repo to GitHub so the user can quickly navigate to the full project.

Challenges: When provided the project, there was only HTML and CSS code not making the project very functional. By adding JavaScript and API's I was able to make the project interactive with all the above features, ensuring the data was pulled directly from the GitHub profile.

Project Highlights

Bio Information: Created JavaScript using an API to automatically pull the information from the GitHub profile for the users name, bio, location and number of repos. If the GitHub profile is updated, it will automatically update the website.

Search Feature: Created a search box that helps the user easily find a repo. When the user inputs a letter or string of letters into the search, it will only display repos that contain that input.
