Unplugged

The client provided PSD mock-up files for mobile, tablet and desktop designs to be built into a responsive website. Built from HTML and CSS, I used clean code and commenting allowing other team members an easy to follow code for quick future updates and troubleshooting.

I implemented CSS Flexbox for a fluid responsive design between mobile, tablet and desktop views. Additionally, an iframe was used to incorporate the google map location on the about page.
