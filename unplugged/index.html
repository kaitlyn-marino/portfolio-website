Unplugged

The client provided PSD mock-up files for mobile, tablet and desktop designs to be built into a responsive website. Built from HTML and CSS, I used clean code and commenting allowing other team members an easy to follow code for quick future updates and troubleshooting.

I implemented CSS Flexbox for a fluid responsive design between mobile, tablet and desktop views. Additionally, an iframe was used to incorporate the google map location on the about page.

PSD Conversion: Designed the mobile, tablet, and desktop versions to match the provided mock-ups. Ensured all additional requirements, such as links and references, were met.

Flexbox Layouts: Implemented wrapped column layouts using CSS Flexbox to allow the number of columns displayed to change based on the size of the screen.

Forms: Implemented a fillable form for email sign-up with fully stylized CSS.

User Experience Enhancements: An iframe was used to enhance the user experience by showing the location of the facility. Reference links for telephone numbers and email addresses were added to enhance the user experience.
