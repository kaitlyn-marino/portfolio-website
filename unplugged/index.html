Project Details

Goal: Create a functional website adhering to mock-ups and all feature requirements.

Role: Built a responsive website using HTML, CSS, responsive design and flexbox to match provided mock-ups.

Features: Used clean code and comments allowing other team members an easy to follow code for quick future updates and troubleshooting, used iframes to incorporate location features, and used flexbox for a fluid responsive design.

Challenges: The client wanted specific views for mobile, tablet, and desktop versions. By having the client provide mock-ups of each view, I was able to match the mock-ups by using flexbox and implementing responsive design.

Project Highlights

Flexbox Layouts: Implemented wrapped column layouts using CSS Flexbox to allow the number of columns displayed to change based on the size of the screen.
