Guess the Word Game

8 chances to guess the correct word

Tools Used

JavaScript

GitHub

Tech Stack

Project Details

Goal: Create a game where the player has 8 chances to guess the correct word.

Role: Created the JavaScript for the game to function and additional JavaScript to enhance the player's experience.

Features: The word is initially displayed with dots and then switches to the correct letters as they are guessed, the game counts down the number of guesses the player has left, displays the guessed letters, displays messages to the player depending on the input, and resets the game if they want to play again.

Challenges: The project initially was only coded and designed with HTML and CSS. By adding JavaScript and using functions and methods, I was able to create all the features listed above and ensure the player has a seamless user experience.

Project Highlights

Word Features: At the start of the game, the word is displayed in dots letting the player know how long the word is. Once a correct letter is guessed, the letter is revealed in the correct position.

Custom Messages: Created custom messages based on the letter input to help assist the player.

Countdown Feature: A countdown feature was added to show the player how many more guesses they have left. If a correct letter is guessed, the countdown number remains the same. When an incorrect letter is guessed the number goes down by 1. If the player runs out of guesses it will flip to a game over message.

Game Over: Once the allowed number of guesses has been reached, the messages will display a game over message including what the correct word is. It will also switch to a Play Again button which resets the whole page with a new word.

Player Won: If the player guesses the correct word in the allotted amount of guesses, it will display a congratulations messagae and flip the guess button to a Play Again button.
