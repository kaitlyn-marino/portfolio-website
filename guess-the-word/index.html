    Guess the Word Game

    8 chances to guess the correct word

    Tools Used

    JavaScript

    GitHub

    Tech Stack

    Project Details

    Goal: Create a game where the user has 8 chances to guess the correct word.

    Role: Client provided the HTML and CSS. I created the JavaScript from scratch to create the user experience.

    Features: The word length is displayed with dots, it counts down the number of guesses the user has left, displays the letters already guessed, and reveals the correct letters in the word.

    Challenges: The client wanted individual messages to be displayed based on what the user inputs. Using functions and methods, I was able to display messages if the letter was already used, if the letter was not valid, and if more than one letter was input.
