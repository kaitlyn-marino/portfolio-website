    Project Details

    Goal: Create a wedding website that pulls in the colors from their Save the Dates and has an exceptional user experience.

    Role: Designed mock-ups, coded the HTML, and stylized the CSS. As they move forward in their planning process, I will be completing the pages currrently marked as "Coming Soon".

    Features: Features an elegant and simple layout that fits their wedding style and encompasses all the details the guests need to know for the wedding, utilized responsive design as many guests will view the website on many different platforms, and incorporated iframes for easy directions.

    Challenges: The client wanted to ensure the website was easy to use on mobile, tablet and desktop and was also easy to use for the older generations that would be attending. By using responsive design I was able to stylize the website on all platforms creating a seamless user experience. Additionally I designed a simple layout making the website easy to navigate for the older generation who may not be as tech savy.
